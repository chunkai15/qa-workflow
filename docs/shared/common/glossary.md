# Everfit Glossary

> Cross-squad terminology. Agent: reference this when encountering unfamiliar terms.

| Term | Definition |
|---|---|
| Everfit | SaaS platform connecting personal trainers with clients |
| Marketplace | Public-facing storefront where clients discover and purchase trainer packages |
| Trainer | Service provider who creates packages and delivers sessions |
| Client | End user who purchases packages and consumes sessions |
| Package | Bundled offering created by trainers (sessions + content) |
| Session Credit | Unit of value used to book sessions |

_Expand as new terms are encountered._
