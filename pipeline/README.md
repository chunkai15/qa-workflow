# Pipeline — Automation Engine

## Structure
```text
pipeline/
  types.ts        ← Shared TypeScript types
  workspace.ts    ← Workspace path utilities
```

## How Generated Code Fits In
Playwright tests generated by WF-4 (`qa-auto-test-generator`) are saved to project folders, not here:
`projects/{squad}/{project}/automation/runs/{run-id}/`

The generated test files import types and path utilities from this central `pipeline/` directory. This keeps the engine infrastructure separated from the project-specific execution code.
